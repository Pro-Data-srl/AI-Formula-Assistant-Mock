# User Prompt zum Testen des Formel-Editors

## Aufgabe: Datum, String verknüpfen und "viel" anhängen, falls Menge größer 0 ist

Bitte erstelle mir eine Formel, die das Erstellungsdatum in Tag und Monat (letzteres ausgeschrieben) angiebt und dann mit bindestrich und artikel name verbindet und falls Menge größer 0 ist, sollte auch noch "viel" mit bindestrich angehängt werden
